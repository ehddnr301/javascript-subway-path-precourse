# 🚀 구현 기능 목록

## 초기설정

- [x] 첫 화면 구성
- [x] 프로그램 시작 시 역, 노선, 구간 데이터를 초기 설정 해야 한다.

```

- 2호선: 교대 - ( 2km / 3분 ) - 강남 - ( 2km / 3분 ) - 역삼
- 3호선: 교대 - ( 3km / 2분 ) - 남부터미널 - ( 6km / 5분 ) - 양재 - ( 1km / 1분 ) - 매봉
- 신분당선: 강남 - ( 2km / 8분 ) - 양재 - ( 10km / 3분 ) - 양재시민의숲
```
